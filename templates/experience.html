EDS App Backend Robust and Scalable Architecture for Service Stations

The backend of EDS App is built with ASP.NET Core, following a hexagonal architecture to ensure clean, decoupled, and maintainable code. It uses JWT for secure token-based authentication and exposes REST APIs documented with Swagger. The solution is optimized for production using Redis for caching, Docker for containerization and portable deployment, and AWS as the cloud platform. GitHub Actions automates the CI/CD pipelines. All development and orchestration are handled through Visual Studio, providing high productivity and full control over the development lifecycle. My role involved contributing to the full development lifecycle as part of a collaborative team—designing the architecture, implementing backend and mobile services, configuring automated testing and CI/CD pipelines, and ensuring security and cloud readiness.
